# 🌿 Paradise Nursery - Shopping Application

## 📌 Project Overview

Paradise Nursery is a responsive front-end shopping application that allows users to browse a variety of houseplants and manage a shopping cart. The application is built using **React**, **Redux Toolkit**, and **React Router**.

Users can explore different plant categories, add items to the cart, and modify quantities before checkout.

## ✨ Features

### 🏠 Landing Page

* Background image
* Company name and description
* "Get Started" button to navigate to products

### 🌱 Product Listing Page

* Displays 6+ houseplants
* Categorized into:

  * Indoor Plants
  * Medicinal Plants
  * Flowering Plants
* Each plant shows:

  * Image
  * Name
  * Price
  * Add to Cart button

### 🛒 Shopping Cart Page

* Displays all added plants
* Shows:

  * Total number of items
  * Total cost
* Features:

  * Increase / Decrease quantity
  * Delete item
  * Continue shopping button
  * Checkout button (Coming Soon)

### 🔝 Header

* Navigation between pages
* Dynamic cart icon with item count

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/shiva200505/plant_project.git
cd plant_project
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally

```bash
npm run dev
```
